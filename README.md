# Rushikesh Chamle - 3D Interactive Web Portfolio

Welcome to the source code for my personal portfolio! This is a modern, highly interactive, and completely responsive full-stack developer portfolio built with React, Vite, and Three.js.

## ✨ Key Features
- **3D Physics-based Tech Stack**: An interactive canvas built with `@react-three/fiber` and `@react-three/rapier` featuring collision-enabled hovering tech bubbles (C, C++, Python, Java, MySQL).
- **Smooth GSAP Animations**: Features high-end scroll and hover animations using `gsap` and `ScrollSmoother`.
- **Advanced Text Splitting**: Beautiful text reveal effects leveraging `SplitText`.
- **Custom Aesthetic**: A hand-crafted, fully responsive dark-mode styled UI with carefully tuned typography and spacing.
- **Dynamic Projects & Career Timelines**: Interactive sections designed to brilliantly showcase my education, robust skill set, and professional journey.

## 🛠️ Built With
- **Framework**: [React 18](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **3D Rendering**: [Three.js](https://threejs.org/) + [React Three Fiber](https://github.com/pmndrs/react-three-fiber)
- **Physics Engine**: [Rapier](https://rapier.rs/)
- **Animations**: [GSAP](https://greensock.com/gsap/)
- **Styling**: Vanilla CSS

## 🚀 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed along with `npm`.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Rushi22007/Rushi-Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Rushi-Portfolio
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running Locally
To start the Vite development server, run:
```bash
npm run dev
```
Then, open [http://localhost:5173/](http://localhost:5173/) in your web browser.
